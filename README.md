# AutonomousFlight

##Ubuntu Install:
1. run install.sh
2. run makeProject.sh

##Mac Install:
1. run install.sh
2. install macports
3. install gcc with 
```bash
sudo port install gcc49
```
*you may also need to type the following as needed

```bash
sudo port -f activate libedit
sudo port -f activate zlib
``` 
4. run makeProjectOSX.sh
